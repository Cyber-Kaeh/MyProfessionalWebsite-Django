## Private Cloud

A self-hosted Nextcloud instance running on Raspberry Pi 4 hardware, Ubuntu Server OS, and Nextcloud's Docker AIO

### Features

- Hardware and OS agnostic file syncing
- Remote Access
- Multiple user accounts
- Talk, text, and video for users
